# Security Policy

## Supported Versions

The latest version on the `main` branch and the most recent stable release are currently supported with security updates.

Older versions may not receive fixes.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not open a public GitHub issue**.

Instead, report it privately by contacting the maintainer through GitHub or by opening a private security advisory if that feature is available for this repository.

Please include:

* A description of the issue.
* Steps to reproduce it.
* Any proof-of-concept code or screenshots (if applicable).
* The version or commit hash affected.

I will investigate the report as quickly as possible and work to provide a fix. Responsible disclosure is appreciated.
